Este es un sitio de prueba para verificar el widget de WhatsApp.

Sitio de Prueba - Manuel Solis

El widget de WhatsApp debe aparecer como un boton flotante en la esquina inferior derecha.

Checklist

  • El boton verde de WhatsApp aparece abajo a la derecha
  • Al hacer hover, muestra el microcopy de consentimiento
  • Al hacer click, abre WhatsApp con mensaje prellenado
  • Terminal del dev server muestra POST /api/consent 200
  • No hay errores CORS en la consola del browser (F12)
